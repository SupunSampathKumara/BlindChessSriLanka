The Blind Chess Project


Blind Chess Sri Lanka is an innovative, fully accessible chess application specifically crafted to empower visually impaired and partially sighted individuals to enjoy the game of chess independently. Built with robust audio feedback and intuitive keyboard navigation, the application translates the complex visual board state into clear, descriptive voice announcements.

Key features include coordinate-based jumps, real-time voice guidance on square content, legal move readouts, and sound effects for moves and captures. By adhering strictly to web and software accessibility guidelines, Blind Chess Sri Lanka creates an inclusive gaming environment, bridging the gap between digital chess and vision accessibility.
